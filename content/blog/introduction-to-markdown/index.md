### Headings
<br/>

```markdown
    # heading1
    ## heading2
    ### heading3
    #### heading4
    ##### heading5
    ###### heading6
``` 
<br/>

### Links
<br/> 

```markdown
    [name of link](https://google.com)
```
<br/>

### Images
<br/>

```markdown
    !['alt'](http://google.com)
```
<br/>

### Bold, Italics
<br/>

```markdown
    **bold**
    *italics*
    ***bold-italics***
```
<br/>

### Lists
<br/>

```markdown
    Ordered
        1 first
        2 second
        3 third
    Unordered
        - first
        - second
        - third
```
<br/>

### Blockquote
<br/>

```markdown
    > Hello
    > World
```
<br/>

### StrikeThrough
<br/>

```markdown
    ~Hello world~~
```
